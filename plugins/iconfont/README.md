# iconfont 图标搜索

> 目前默认图片格式为 300 \* 300 png 格式
> 开源地址: [https://github.com/mohuishou/utools](https://github.com/mohuishou/utools) 还有 chrome 历史记录搜索, vscode 历史搜索等插件等你发现

## 使用方法

- 输入关键词搜索
- enter 下载 png 图片
- command/ctrl + enter 复制图片到剪切板
- option/alt + enter 浏览器打开 iconfont
