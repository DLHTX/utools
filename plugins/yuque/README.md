# 语雀搜索

## 特性

- 快速搜索语雀知识库
- 支持复制为 markdown，支持语雀区块语法（替换为对应的 HTML 文件）

## 使用

### 1. 设置 token

1. 参考文档获取 token [https://www.yuque.com/yuque/developer/api#785a3731](https://www.yuque.com/yuque/developer/api#785a3731)
2. utools 输入框输入 `语雀设置`
3. 将你获得的 token 复制在输入框，并且回车

### 2. 搜索

1. utools 输入框输入 `语雀搜索`
2. 选择你想要的结果，回车
3. 选择前往页面，或者复制为 markdown
