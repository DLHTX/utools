# chrome-history

utools plugin, 搜索 chrome 浏览器历史记录，支持多个关键字匹配

## 下载地址

[点击下载](https://github.com/mohuishou/utools/releases/tag/1.0.0)

**如果觉得不错点个 star ❤️**

## key

`ch`: 搜索
`ch-setting`: 搜索设置
`enter`: 访问链接
↑↓ : 切换条目
