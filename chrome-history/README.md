# chrome-history

utools plugin, 搜索 chrome 浏览器历史记录，支持多个关键字匹配

## 使用方法

- **如果觉得不错点个 star ❤️，开源地址(还有许多其他插件等待你发现哦): [https://github.com/mohuishou/utools](https://github.com/mohuishou/utools)**
- 输入 `ch` 触发搜索，支持多个关键词筛选，使用空格间隔
