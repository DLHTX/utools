# chrome-history

utools plugin, 搜索 chrome 浏览器历史记录，支持多个关键字匹配

## 使用方法

- **如果觉得不错点个 star ❤️, [https://github.com/mohuishou/utools](https://github.com/mohuishou/utools)**
- 输入 `ch` 触发搜索，支持多个关键词筛选，使用空格间隔
- 输出 `chs` chrome 目录
